Image swap

Χρησιμοποιώντας την υπηρεσία picsum εναλλάξτε την παρακάτω εικόνα

Tip

Επιλέξτε την εικόνα (με id="image-swap") και στο 'mouseenter' event αλλάξτε το src attribute [ELEMENT.getAttribute("ATTRIBUTE NAME")] να δείχνει την εικόνα "https://picsum.photos/200/300". Στη συνέχεια αποκαταστήστε την με το 'mouseleave'.

Ξεκινήστε με:
const img = document.querySelector('#image-swap');...
